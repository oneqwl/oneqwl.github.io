js知识

  • js的组成

    *   BOM(浏览器对象模型)，DOM（文档对象模型），ECMAscript（js语法核心）

  • js中有两种数据类型

    *   基本数据类型：string（字符串），number(数值)，boolean（布尔类型），null（空），undefined（未定义）
    • 复杂数据类型：Qbject（对象类型），Function（函数类型），Array（数组类型），Date（日期类型），Regexp（正则表达式）

  • 判断数据类型（关键字：typeof）

    *   语法：typeof 数据。例如：

    <span class="hljs-built_in">console</span>.log(<span class="hljs-keyword">typeof</span> <span class="hljs-literal">null</span>);    <span class="hljs-comment">//object</span>

    console.log(typeof “demo”); //字符串类型
    console.log(typeof undefined); //undefined
    console.log(typeof [1, 2, 3]); //object
    console.log(typeof function () {
    }); //function
    `

    • typeof的返回值是string类型的。例如：
      `console.log(typeof typeof null);  //string

  • js中的数据按照存储的内容不同可以分为

    *   值类型(存储在栈中)：保存的是具体的数据。

    基本数据类型都是值类型

    *   引用数据类型（存储在堆中）：保存的是引用，这个引用指向的是内存中的一块具体的数据

    复杂数据类型都是引用类型
